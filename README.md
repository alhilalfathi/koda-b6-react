# Coffee Shop Ecommerce
Web app coffee shop ecommerce yang berisikan informasi mengenai menu, harga, maupun diskon. web ini dibuat menggunakan react dan styling menggunakan tailwindcss.

## Cara menggunakan
1. Download semua resource di repo ini
2. Buka terminal
3. Ketik perintah npm run dev
